## 技术推演路线

静态页面抓取 → 动态渲染抓取 → 框架与分布式 → JS 逆向

### 项目对比：从 Requests 到 Selenium

| 维度     | 豆瓣电影 Top250        | 网易新闻首页                         |
| :------- | :--------------------- | :----------------------------------- |
| 采集方式 | Requests 直接请求 HTML | Selenium 模拟浏览器渲染              |
| 加载方式 | URL 分页（静态翻页）   | 滚动 + 点击"加载更多"（动态加载）    |
| 核心难点 | 模块化拆分、数据去重   | 动态内容触发、按钮定位、反自动化检测 |
| 存储模块 | 字段写死，与业务耦合   | 通用模块，零改动跨项目复用           |
| 反爬策略 | UA 轮换 + 请求延时     | UA 伪装 + 无头模式 + 自动化特征隐藏  |
